# API Reference

Fabrica currently exposes a deliberately small public API:

| API | Description |
| --- | --- |
| `html` | Tagged template helper for immutable HTML fragments. |
| `css` | Tagged template helper for immutable CSS fragments. |
| `HtmlTemplate` | Runtime object returned by `html`. |
| `CssTemplate` | Runtime object returned by `css`. |
| `FabricaTemplate` | Shared immutable template metadata shape. |
| `TemplateValue` | Interpolation value type accepted by template helpers. |
